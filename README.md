# Asset Management

A digital asset marketplace for buying and selling digital products. Built as a learning project with Next.js.

## About

This platform allows users to list and purchase digital assets. Users can add their digital products such as images, files etc, which need admin approval before being listed on the marketplace. Other users can then browse and purchase these approved assets.

## Features

- Add and list digital assets
- Admin approval system for listings
- Browse and purchase digital assets
- User authentication

## Tech Stack

- Next.js
- TypeScript
- PostgreSQL
- Drizzle ORM
- shadcn/ui
- Tailwind CSS
- Better Auth
